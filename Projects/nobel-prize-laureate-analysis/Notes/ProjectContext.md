# Project Context

This project was completed as part of a DataCamp learning exercise and adapted for portfolio organization.

The analysis explores Nobel Prize laureates from 1901 to 2016, with a focus on gender representation, country-level dominance, repeat winners, category-level trends, and winner age over time.

## Skills Demonstrated

- Data loading and inspection
- Exploratory data analysis
- Grouping and aggregation
- Time-based feature engineering
- Data visualization
- Categorical trend analysis
- Historical and demographic analysis
- Jupyter Notebook organization
