# Project Context

This project was completed as part of a DataCamp learning exercise and adapted for portfolio organization.

The analysis explores Super Bowl game outcomes, television viewership, advertising trends, and halftime show performances.

## Skills Demonstrated

- Data loading and inspection
- Exploratory data analysis
- Data cleaning
- Data visualization
- Joining related datasets
- Trend analysis
- Sports and media analytics storytelling

Dataset source: DataCamp project/course dataset.
