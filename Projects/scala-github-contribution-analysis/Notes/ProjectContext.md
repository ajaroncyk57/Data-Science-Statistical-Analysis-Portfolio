# Project Context

This project was completed as part of a DataCamp learning exercise and adapted for portfolio organization.

The analysis explores the contribution history of the Scala open-source project using pull request and file-level change data mined from GitHub.

## Skills Demonstrated

- Data loading and inspection
- Data cleaning
- Combining datasets
- Date and time parsing
- Exploratory data analysis
- Grouping and aggregation
- Developer contribution analysis
- Open-source repository analysis
- Data visualization
- Jupyter Notebook organization

## Dataset Source

Dataset source: DataCamp project/course dataset.

Raw dataset files are included only if redistribution is permitted.
