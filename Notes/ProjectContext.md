# Project Context

This project was completed as part of a DataCamp learning exercise and adapted for portfolio organization.

The analysis explores Android app market trends using Google Play Store app metadata and user review sentiment data.

## Skills Demonstrated

- Data loading and cleaning
- Exploratory data analysis
- Data visualization
- App category and pricing analysis
- Sentiment analysis using review polarity scores
- Jupyter Notebook organization

## Dataset Source

Dataset source: DataCamp project/course dataset.

Raw dataset files may not be included in this repository unless redistribution permissions are confirmed.
